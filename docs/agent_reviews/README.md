# Agent Reviews

This folder is the shared review queue for Codex, Claude, and Keith.

Use it for durable suggestions, product critiques, implementation reviews, and
handoffs that should survive chat-session boundaries.

Rules:

- No PHI.
- No credentials.
- No real patient identifiers.
- No real claim numbers.
- No client-sensitive screenshots.
- Keith approves before suggestions become implementation work.

Subfolders:

- `claude/`: review notes from Claude.
- `codex/`: review notes from Codex.
- `resolved/`: approved and acted-on notes.
